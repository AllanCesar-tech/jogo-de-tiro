# Projeto Space Shooter em JavaScript

<p style="text-align: justify;">Este projeto é uma <strong>iniciativa pessoal</strong> para aprimorar minhas habilidades em <strong>JavaScript</strong>, focando na criação de jogos. Meu objetivo inicial era <strong>recriar um jogo do estilo space shooter</strong>, explorando conceitos de programação, animações e lógica de jogo.</p>
  
<p style="text-align: justify;">No entanto, ao longo do desenvolvimento, <strong>descobri que tenho uma maior afinidade com back-end</strong> e outras áreas da tecnologia. Por isso, decidi dar uma pausa no projeto para focar nessas áreas que considero mais importantes. Assim, a criação do jogo acabou se tornando um <strong>hobby</strong>, uma atividade divertida e desafiadora para aprimorar minhas habilidades de programação sem perder o prazer pelo aprendizado.</p>

<p style="text-align: justify;">Este projeto é uma <strong>experiência contínua</strong>, e espero retomar o desenvolvimento do jogo futuramente, além de explorar novas possibilidades na programação de jogos e outros projetos.</p>


## Tecnologias Utilizadas

- JavaScript
- HTML

## Status do Projeto

- Em pausa temporária
- Hobby e fonte de aprendizado contínuo
